# Royal Luxury Rent a Car

A Next.js 15 website scaffolded for `royalluxuryrentacar.com`.

## Features

- SEO metadata for luxury car rental search terms
- Hero slider with 3-second autoplay
- Fleet cards with call and WhatsApp actions
- Contact form with a generated email template
- Footer gallery image modal with navigation
- Scroll reveal animations

## Setup

1. `npm install`
2. `npm run dev`

## Notes

- Images are stored in `public/images`
- Contact form is currently client-only and displays a generated email template.
